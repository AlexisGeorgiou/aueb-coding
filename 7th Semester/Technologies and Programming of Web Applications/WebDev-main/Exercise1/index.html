Η πλατφόρμα Gnosis

Η διαδικτυακή πλατφόρμα Gnosis, είναι μια σύγχρονη διαδικτυακή υπηρεσία μάθησης υψηλού επιπέδου που παρέχει στους συμμετέχοντες υπηρεσίες και υλικό αντίστοιχου επιπέδου με τα πανεπιστήμια της χώρας. Είναι μιας πρώτης τάξεως ευκαιρία για να έρθετε σε μια πρώτη επαφή με τεχνολογίες που χρησιμοποιούνται σε όλους τους χώρους εργασίας ανά τον κόσμο και να επεκτείνετε τις γνώσεις σας σε μια σειρά από θεματικές ενότητες. Επίσης, με την εγγραφή σας σε οποιοδήποτε μάθημα, έχετε την ευκαιρία να μάθετε και να αλληλεπιδράσετε με καθηγητές κορυφαίου επιπέδου. Τέλος, μέσα από την συμμετοχή σας στα ολιγάριθμα διαδικτυακά τμήματα, έχετε την δυνατότητα να είστε σε άμεση και διαρκή επικοινωνία με τους καθηγητές σας, οι οποίοι είναι πάντοτε πρόθυμοι να λύσουν κάθε σας απορία και να σας καθοδηγήσουν ούτως ώστε να πετύχετε τα καλύτερα δυνατά αποτελέσματα.

Παρακολουθήστε Διαδικτυακά Μαθήματα

Στην πλατφόρμα μας θα βρείτε πάνω από 60 μαθήματα από 5 διαφορέτικες θεματικές ενότητες που πραγματοποιούνται από αξιόλογους και έμπειρους καθηγητές. Οι κύριες θεματικές περιοχές της υπηρεσίας μας, χωρίζονται σε πολλά επιμέρους γνωστικά αντικείμενα, από τα οποία μπορείτε να επιλέξετε αυτά που σας ενδιαφέρουν περισσότερο. Περισσότερες πληροφορίες για το κάθε γνωστικό αντικείμενο μπορείτε να βρείτε στην αντίστοιχη σελίδα.
