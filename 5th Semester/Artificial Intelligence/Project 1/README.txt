----------README ΤΗΣ ΕΡΓΑΣΙΑΣ ΤΩΝ ΦΟΙΤΗΤΩΝ 3180021, 3180027, 3180094----------
Η εργασία δημιουργήθηκε με την βοήθεια του προγραμματιστικού περιβάλλοντος Visual Studio Code
 και έγινε με την χρήση της γλώσσας JAVA. Περιέχονται τα 4 εξης αρχεία:
1. Main.java
2. GamePlayer.java
3. Move.java
4. Board.java

Η μεταγλώτισση του προγράμματος γίνεται φυσικά με την εντολή
javac *.java
στο CMD με το ανάλογο path
Μετά την μεταγλώτισση προκύπτουν τα ακόλουθα 4 αρχεία.
1. Main.class
2. GamePlayer.class
3. Move.class
4. Board.class

Η εκτέλεση του προγράμματος απαιτεί την εκτέλεση του Main.class το οποίο γίνεται με την εντολή:

java Main

στο CMD με το ανάλογο path
